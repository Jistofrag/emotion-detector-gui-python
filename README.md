# Emotion Detector GUI

This is a Python project that checks the emotion of any text you type (positive, negative, neutral) using a modern GUI.

## How to Use
1. Install Python 3 and pip.

2. Install the needed libraries: 
pip install vaderSentiment customtkinter

3. Open `emotion_detector.py` and run it:
python emotion_detector.py

4. Type any sentence in the app and see the emotion result!

## Features
- Easy-to-use window (GUI)
- Shows emotion as emoji and percentage
- Works offline
